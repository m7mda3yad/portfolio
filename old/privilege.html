Privilege Education App - Project Details

Project Overview

View on Google Play

The "Privilege Education App" is a mobile application targeting university students. When a student opens the app, they are presented with a list of universities. Within each university, they can navigate to different faculties and then to courses offered by those faculties. Students can purchase courses online, either as a whole course or a single lesson.

Key Features

  • University and Faculty Navigation
  • Online Course Purchase
  • Lesson-by-Lesson Purchase
  • Course Viewing and Interaction
  • Chat Functionality for Student-Teacher and Student-Student Interaction
  • File Download for Course Materials

Admin and Teacher Dashboards

  • Admin Dashboard for Full Control
  • Teacher Dashboard for Course Management
  • Communication Tools

Technologies Used

  • HTML, CSS, JavaScript
  • Bootstrap
  • PHP and Laravel Framework
  • MySQL Database
  • Mobile App Development Framework ( Flutter )
